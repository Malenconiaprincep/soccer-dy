# 足球经理对战 · 设计规格与 Prompt 文档

本文档与工程内 `src/platform/constants.ts`（逻辑分辨率 **750×1624**，竖屏）对齐，可直接复制到 AI 设计或素材平台使用。

## 画布与尺寸（硬约束）

- **逻辑设计分辨率（手机竖屏）**：**750×1624 px**（宽×高）。全屏缩放采用 **contain** 思路：关键 UI 与交互落在安全区内。
- **安全区**：四边各 **≥56 px**（相对逻辑稿）为装饰区；核心按钮与文字勿进入距边 **<48 px** 风险带（刘海、圆角、底部横条）。
- **触控热区**：最小 **88×88 px**。
- **纹理**：单张建议 **≤2048×2048**，长边优先 **≤1024**（中低端 GPU 与包体）。

## 风格 bible（每条资产 prompt 前粘贴；全局 Q 版）

### 中文风格锚（可复制）

> 移动端竖屏足球经理对战小游戏 UI + 场景素材，画布比例 **750:1624**，**面向手机全屏显示**。美术为**统一 Q 版 / 萌系**（**二～三头身**角色比例、**大头小身**、圆滚滚几何、可爱但不低幼），UI 与场景同为 **Q 版卡通**：**粗描边 2～3px**（按设计稿 1x）、色块分明、**略夸张圆角**与「鼓鼓的」体积感，轻渐变 + 柔和投影，**高对比可读**。**非写实、非肌肉写实**，不要真人比例。**主色**：草绿 `#2D8F47`、深绿 `#1B5E32`、面板深灰蓝 `#1E2430`、高光 `#F5F7FA`、强调色琥珀 `#F5A623`（CTA 与比分高亮）。**禁止**：水印、文字乱码、糊边、血腥暴力、**写实球场照片**。**输出**：透明背景 PNG（UI/角色），或指定底色 PNG（全屏背景），边缘干净无杂边。

### 英文风格锚（部分模型只吃英文时使用）

> Mobile portrait football manager PvP mini-game assets for **phone fullscreen**, **750x1624** logical layout. **Consistent chibi / super-cute Q-style**: **2–3 heads tall** characters, **big head small body**, rounded chunky shapes, friendly but not childish. UI and environment match **cute cartoon**: **bold 2–3px** outlines, saturated readable colors, bubbly rounded corners, subtle gradients and soft shadows. **Palette**: pitch green `#2D8F47`, deep green `#1B5E32`, panel `#1E2430`, light `#F5F7FA`, accent `#F5A623`. **Not** photorealistic, **not** realistic anatomy. **No** watermark, no illegible text. **Export**: crisp PNG, transparent where noted.

### 负面词建议（平台若有单独栏）

`photorealistic, realistic anatomy, muscular, gore, watermark, blurry, lowres, extra fingers, deformed ball, horror, gritty`

## 命名与导出约定

- **文件名**：全小写 + 下划线，与上表 `资产 ID` 一致；本工程请放入 **`src/assets/game/`**（由 Vite 打进各端包体）。
- **竖屏像素**：主菜单底图建议导出为 **750×1624**；球场贴图与场内逻辑框一致为 **660×920**（见 `match.ts` 中 `PITCH_*`）。若 AI 平台只能出横向固定画布（常见 **1536×1024**），请把 PNG 放进 `src/assets/game/` 后在本机执行 **`pnpm run bake:portrait`**（依赖已安装的 ImageMagick `magick`），脚本会按上述比例**居中裁切**成竖版，避免在引擎里非等比拉伸。
- **字体**：位图字需可商用授权；系统字仅作设计示意。
- **队徽与队名**：第一期使用**虚构名称与几何队徽**，避免真实俱乐部商标。
